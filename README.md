# Raypong

A simple pong game built with [Odin](https://odin-lang.org/) and [Raylib](https://www.raylib.com/).

## Building

See [BUILD.md](BUILD.md) for build instructions.

## Running

After building, you can run the game directly from the `bin/` directory:

**Windows:**
```cmd
bin\raypong.exe
```

**Linux:**
```bash
./bin/raypong
